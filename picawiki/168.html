altstore 安裝嗶咔ios Ipa自簽

8 個月 ago
wikimanger
1 minute

本流程只考虑电脑和 ios 手机使用 Altstore 的自签，自签时效只有 7 天，但只有第一次比较麻烦，之后刷新只要有电脑就能很快完成。

更新了第三方視頻教程 教程來自群組用戶分享
（第三方教程和本站無關，若提示付費等等請不要付費，真的非常簡單）


由于 wi-fi 连结因人而异会有失败的情况，本流程只使用数据线连接电脑和手机。
如果不能成功也请不要纠缠我谢谢。

  1. 首先在电脑上打开 Altstore 官网 https://altstore.io/
  1. 选择 Windows（beta）下载压缩包(Mac同理)
  2. 解壓压缩包
  3. 选择 setup 安装 altstore
  4. 安装目录，最好全英文，然后 next，next 直到安装完成。
  5. 打开之前下载的文件，打开图中标蓝的

之后会提示要你安装最新的 icloud

請注意！！
不要從window應用商店下載iTunes，如果下載了請卸載
請前往apple官網下載iTunes
點這裏https://support.apple.com/downloads/itunes

icloud 安装完成后，再次打开图中标蓝的程序，程序打开后不会出弹框，而是在下方任务栏
出现一个灰暗的方形图标

选择 install AltStore ，如果右侧没出现你的手机名称，则是你手机没连上电脑，请用数据线
连接电脑，信任电脑后再次点击 install Altstore，应该会出现 iphone 或者之类的选择。
之后会出现认证 apple id 的选项，输入你手机的 apple id 然后认证。

請注意：如果是手機號appid 明明密碼輸入正確，但是提示錯誤
請在手機號前+86
例如
155221223333
修改爲：86155221223333

稍等片刻应该会提示 install 完成，但是暂时不要断开数据线，在手机上打开 altstore app，
首次打开的时候会提示非信任软件，去手机的设置-通用-设备管理内信任此软件后才能打
开。

后点击右上角的用其他应用打开图标，选择用 altstore 打开。

下載IOS 13 請下載舊版
IOS14-15請下載新版
IOS 15.4 安裝完成閃退請看這裏

下载完成后打开，然后点击右上角的用其他应用打开图标，选择用 altstore 打开。


之后就会切到 altstore 内，完成软件的自签。（如果显示找不到，那就是是手机没连接到电
脑，或者电脑的 altstore 没在后台运行）。
